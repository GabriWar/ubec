================================================================================
GUIA SIMPLIFICADO - ADICIONAR MAPEAMENTO MODBUS NO SOMACHINE
================================================================================

MÉTODO MAIS RÁPIDO: Copiar e Colar no SoMachine
================================================================================

1. Abra PROJETO2_com_modbus.smbp no SoMachine

2. Vá até os rungs TEST_M100_SET e TEST_M100_RESET que você já criou

3. SELECIONE os 2 rungs (clique no primeiro, Shift+clique no segundo)

4. COPIE (Ctrl+C)

5. COLE (Ctrl+V) - isso vai duplicar os 2 rungs

6. EDITE o primeiro rung colado:
   - Clique duplo em %M100 → mude para %M101
   - Clique duplo em %MW500 → mude para %MW501
   - Clique duplo no nome → mude para MAP_M101_SET

7. EDITE o segundo rung colado:
   - Clique duplo em %M101 → (já está certo do passo anterior)
   - Clique duplo em %MW501 → (já está certo)
   - Clique duplo no nome → mude para MAP_M101_RESET

8. REPITA os passos 3-7 para:

   ENTRADAS (13 vezes):
   M102 -> MW502
   M103 -> MW503
   M104 -> MW504
   M105 -> MW505
   M106 -> MW506
   M107 -> MW507
   M108 -> MW508
   M109 -> MW509
   M110 -> MW510
   M111 -> MW511
   M112 -> MW512
   M113 -> MW513

   SAÍDAS (10 vezes):
   M200 -> MW600
   M201 -> MW601
   M202 -> MW602
   M203 -> MW603
   M204 -> MW604
   M205 -> MW605
   M206 -> MW606
   M207 -> MW607
   M208 -> MW608
   M209 -> MW609

9. SALVE o projeto

10. COMPILE (F7)

11. TRANSFIRA para o simulador


================================================================================
OU ALTERNATIVA MAIS RÁPIDA:

  - Entradas: FC3, addr 100, qty 14
  - Saídas: FC3, addr 200, qty 10

  - Force MW100 = 1 (ao invés de M100)
  - Force MW200 = 1 (ao invés de M200)

Isso vai funcionar IMEDIATAMENTE sem precisar adicionar rungs!
